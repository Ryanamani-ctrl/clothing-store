## How to Use

1. **Browse Products**: Start on the homepage or navigate to the Shop page
2. **Add to Cart**: Click "Add to Cart" on any product
3. **View Cart**: Click "Cart" in navigation to see your items
4. **Checkout**: Click "Proceed to Checkout" to complete your order
5. **Place Order**: Fill in your details and click "Place Order"

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with flexbox and grid
- **JavaScript**: Vanilla JS for all functionality
- **LocalStorage API**: Client-side data persistence
## Setup Instructions

1. Download all files to the same directory
2. Ensure product images are in the same folder
3. Open `index.html` in a web browser
4. No server or build process required!
